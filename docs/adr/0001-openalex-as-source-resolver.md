# OpenAlex as the Source Resolver

The audit verifies that each cited **Source** actually exists in the world — defending against fabricated/hallucinated citations. We chose **OpenAlex** as the single resolution provider for v1, despite Crossref being the canonical metadata registrar for DOI-bearing works. OpenAlex covers DOI-bearing works (via its Crossref import), preprints (via arXiv), books, theses, and grey literature behind one free API with no API key required, which avoids building separate Crossref + arXiv integrations and dramatically reduces `UnverifiableSource` Findings for non-DOI works. The trade-off accepted: OpenAlex re-publishes Crossref metadata, so DOI-bearing entries are read one hop downstream of the registrar — acceptable because OpenAlex's update cadence is short and the audit's failure mode is title/author mismatch (which a stale republished record would still detect).
